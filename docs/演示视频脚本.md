## 📋 详细操作步骤

### 一、准备（录制前）

1. 手机打开浏览器，访问 **http://106.13.186.155:7078**
2. 确认页面加载成功（看到「吉小图」欢迎界面）
3. 打开手机录屏功能
4. **开始录制** → 停留 3 秒展示页面

---

### 二、片头（0:00-0:08）— 8秒

> 此时录制已开始，直接展示手机屏幕即可，后期加字幕

**画面**：展示 Web 首页（欢迎界面 + 快捷按钮）

**后期字幕**：
```
吉小图
图书馆新生AI导览助手
NVIDIA DGX Spark Hackathon
```

---

### 三、身份识别（0:08-0:22）— 14秒

**操作**：点击底部输入框，输入「你是谁」，点击发送 ✈️

**等待 2-3 秒**，AI 回复出现后停留 2 秒让观众看清楚。

**后期字幕**：「Multi-Agent 协作：Guide Agent 意图识别 + Search Agent 知识检索」

---

### 四、借阅问答（0:22-0:38）— 16秒

**操作**：输入「借阅规则是什么」，点击发送

**等待 3-5 秒**，AI 流式回复逐字出现。文字出现后停留 2 秒。

**后期字幕**：「RAG 混合检索 + Step 3.7 Flash 生成」

---

### 五、拍照导航 — 核心亮点（0:38-0:55）— 17秒

**操作**：
1. 点击底部 📷 相机按钮
2. 对准书架或图书封面拍照（可以提前准备一张图书馆书架照片在相册里，点「从相册选择」）
3. 确认照片后，等待AI识别（显示「正在识别中...」）
4. AI 返回「1F东北，大屏右侧」

**后期字幕**：「Step 3.7 Flash 多模态识别 + 路径规划 Agent」

---

### 六、路径导航（0:55-1:10）— 15秒

**操作**：输入「怎么去期刊阅览室」，点击发送

**等待 3-5 秒**，AI 返回分步导航指引。

**后期字幕**：「Planning Agent：A* 路径规划 + 楼层图导航」

---

### 七、开放时间（1:10-1:25）— 15秒

**操作**：输入「周末开门吗」，点击发送

**等待 3 秒**，展示 AI 回复。

**后期字幕**：「Verify Agent 事实校验，确保回复准确」

---

### 八、结尾（1:25-1:30）— 5秒

**操作**：停止录制

**后期画面**：
```
吉小图 — 让每一位新生不再迷路
吉利学院图书馆团队
杨博 · 杨帅 · 高智洪
```

## 🎨 后期制作建议

| 项目 | 建议 |
|------|------|
| 字幕 | 底部居中，白色字 + 半透明黑色背景 |
| 操作提示 | 圆圈高亮点击位置，持续 0.5 秒 |
| AI 回复 | 可倍速播放等待过程（标注 2x） |
| BGM | 轻快科技风纯音乐，音量调低 |
| 关键帧 | AI 回复文字出现时，短暂放大 |

---

## ⚠️ 注意事项

1. **手机横屏录制**，分辨率 1920×1080
2. **关闭通知**，避免弹窗遮挡
3. **网络稳定**，确保 DGX Spark 服务正常
4. **提前测试**每个查询的响应速度，熟悉等待节奏
5. 如果某个查询响应慢（>15秒），可以剪掉等待部分
6. 拍照功能如果相机权限有问题，用「从相册选择」备选
