# 部署说明文档

## 一、环境要求

### 硬件要求
- **DGX Spark GB10**（推荐）：128GB 统一内存，NVIDIA GPU
- 或任意配备 NVIDIA GPU 的 Linux 服务器

### 软件要求
- Python 3.10+
- CMake 3.20+、GCC 11+
- Git

## 二、快速体验部署（Stepfun API 模式）

无需 GPU，适合快速体验。

```bash

cd Multi-Agent-Library-Guide-System

# 编辑 .env，填入 STEPFUN_API_KEY

```

访问 `http://localhost:8000`

## 三、DGX Spark 本地 GPU 部署

充分利用 DGX Spark GB10 128GB 统一内存，部署 Qwen3.6-35B-A3B MoE 本地推理。

### Step 1: 编译 llama.cpp (CUDA)

```bash
git clone --depth 1 https://github.com/ggerganov/llama.cpp.git
cd llama.cpp
cmake -B build -DGGML_CUDA=ON -DCMAKE_CUDA_ARCHITECTURES=native -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release -j$(nproc)
```

### Step 2: 下载 Qwen3.6 MoE 模型

从 ModelScope 国内源下载（~21GB）：

```bash
pip install modelscope
python3 -c "
from modelscope import snapshot_download
snapshot_download('unsloth/Qwen3.6-35B-A3B-GGUF', local_dir='/opt/models')
"
```

### Step 3: 启动 llama-server

```bash
./build/bin/llama-server \
  -m /opt/models/Qwen3.6-35B-A3B-MXFP4_MOE.gguf \
  -mm /opt/models/mmproj-F16.gguf \
  --jinja -ngl 99 -fa on \
  -c 32768 --parallel 2 --cont-batching \
  -ctk q4_0 -ctv q4_0 -t 8 -tb 8 --mlock \
  --host 127.0.0.1 --port 8080
```

### Step 4: 部署本项目

```bash
git clone https://github.com/finalyb/Multi-Agent-Library-Guide-System.git
cd Multi-Agent-Library-Guide-System

python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt

cp .env.example .env  # 编辑填入 STEPFUN_API_KEY
python scripts/build_kb.py
uvicorn backend.main:app --host 0.0.0.0 --port 8888
```

### Step 5: 端口映射

DGX Spark 内网端口 8888 映射到公网端口。访问公网地址即可。

## 五、架构说明

```
手机浏览器 → 公网 → FastAPI(:8888) → llama-server(:8080)  [文本推理]
                                      → Stepfun API         [多模态识别]
                                      → BM25 关键词检索     [RAG]
```

- **文本推理**：本地 Qwen3.6 MoE GPU 推理（或 Stepfun 降级）
- **多模态**：Stepun Step 3.7 Flash（拍照识别）
- **意图分类/校验**：Stepfun API（快速轻量调用）
- **RAG 检索**：BM25 关键词匹配（104篇文档，即时响应）

## 六、常见问题

| 问题 | 解决 |
|------|------|
| CUDA error: no kernel image | 重新编译，确保 `-DCMAKE_CUDA_ARCHITECTURES=native` |
| OOM 显存不足 | 降低 `--parallel 1`，开启 `-ctk q4_0 -ctv q4_0` |
| 连接失败 | 检查内存 `free -h`，90%+ 是内存溢出 |
| 拍照识别失败 | 系统自动降级为预设位置+关键词查询 |
| 知识库更新 | 修改 JSON 文件后运行 `python scripts/build_kb.py` |
